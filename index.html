Tip 01
活动时间填准确一点
开始和结束时间越准确，分析出的空档就越精确。不确定的话填估计值就好。
Tip 02
地点写商圈或地铁站名
比如「朝阳大悦城」比「朝阳北路某店」更容易找到附近推荐地点。
Tip 03
空档门槛建议 45分钟–1小时
太短很难完成有意义的活动；太长会错过很多填充机会。
Tip 04
DeepSeek API Key 怎么拿
